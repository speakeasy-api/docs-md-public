## <Pill variant="primary"><b>POST</b></Pill> /rest/api/v1/createauthtoken \{#operation-createauthtoken\}

</OperationTitleSection>

<OperationSummarySection slot="summary">

_Create authentication token_

</OperationSummarySection>

<OperationDescriptionSection slot="description">

Creates an authentication token for the authenticated user. These are
specifically intended to be used with the [Web SDK](https://developers.glean.com/web).

Note: The tokens generated from this endpoint are **not** valid tokens
for use with the Client API (e.g. `/rest/api/v1/*`).


</OperationDescriptionSection>
