</OperationTitleSection>

<OperationSummarySection slot="summary">

_List available tools_

</OperationSummarySection>

<OperationDescriptionSection slot="description">

Returns a filtered set of available tools based on optional tool name parameters. If no filters are provided, all available tools are returned.

</OperationDescriptionSection>
